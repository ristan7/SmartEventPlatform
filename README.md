# Event Management Platform

Platform for managing professional events, speakers, locations, and participant registrations.

## Architecture
- Monolithic application
- MVC architecture
- Planned migration to microservices

## Technology Stack
- ASP.NET Core MVC
- .NET 10
- Entity Framework Core
- SQL Server
- Razor Views

## Notes
The application is currently developed as a monolith, using the same technology stack as the provided starter application, with a future goal of transitioning to a microservices architecture.
